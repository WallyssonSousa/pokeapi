# Pokeapi

<p>É um simples projeto em que o usúario digita um número, e aparece o nome e uma imagem do pokemon, podendo ver ele de frente e de costas.</p>

# Como funciona

<p>Os dados vem da api: https://pokeapi.co/</p>
